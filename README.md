# Manuel de Gestion – Veille Médiatique Audiovisuelle

Ce dépôt contient un site statique en français présentant un manuel complet pour concevoir et exploiter un service de veille médiatique audiovisuelle.

## Contenu

- `index.html` – page principale structurée en sections (vision, organisation, processus, outils, qualité, conformité, gestion de crise, roadmap et annexes).
- `styles.css` – feuille de style responsive utilisant la police Inter et une mise en page moderne.

## Utilisation locale

Ouvrez simplement `index.html` dans votre navigateur ou servez le dossier à l’aide d’un serveur statique :

```bash
python -m http.server 8080
```

Ensuite, accédez à `http://localhost:8080` pour consulter le manuel.

## Déploiement

Le projet est prêt pour un hébergement statique (ex. Vercel). Aucun build ou dépendance n’est nécessaire.
